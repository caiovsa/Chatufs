### Dados do Programa

---


  

| **Ano-Período:**   2013-1 |
| --- |
| **Quantidade de Avaliações:**   3 |
| Objetivos: | |
| Ensinar a descrever e a apresentar as informações relevantes contidas em um conjunto de dados. | |
| Conteúdo: | |
| - Uma visão geral da Estatística.- Divisões da Estatística.- Conceitos básicos. Dados e variáveis Estatísticas.- Fases do método/ trabalho estatístico.- Séries estatísticas.- Distribuições de frequências e seus elementos. - Representação de dados por meio de tabelas e gráficos.- Medidas de posição.- Medidas de dispersão.- Medidas de assimetria e curtose.- Regressão - idéias iniciais e interpretação de parâmetros.- Correlação e coeficiente de determinação. | |
| Competências e Habilidades: | |
| Ao final do período o aluno terá condições de descrever e apresentar as informações relevantes contidas em um conjunto de dados. | |
