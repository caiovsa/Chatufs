# Chat UFS (Piloto)
----
Apenas um piloto para o chat UFS que vai ser demonstrado!

Basicamente essa versão vai ser um chat padrão (Com chamada de API e execução de modelo local) que vai falar sobre assuntos gerais e sobre conteudodo portal UFS

* Chat usando a API do gemini e com algum modelo local
* Leve RAG sem personalização, com os "documentos" fixos! No caso os documentos são oque consegurimos fazer scrap do portal UFS
